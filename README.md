# Cu Yemekhane

Cukurova University Cafeteria Menu Application.

## Description

This project is a Next.js application designed to display the daily food menu of Cukurova University. It scrapes data from the university's official website and presents it in a user-friendly interface.

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Lucide React
- Cheerio (for web scraping)
- Radix UI

## Installation

1. Clone the repository.
2. Install dependencies:

```bash
npm install
# or
pnpm install
```

## Usage

Run the development server:

```bash
npm run dev
# or
pnpm dev
```

Open http://localhost:3000 in your browser to see the result.

## Scripts

- dev: Runs the development server.
- build: Builds the application for production.
- start: Starts the production server.
- scrape: Scrapes the latest menu data.

```bash
npm run scrape
```
